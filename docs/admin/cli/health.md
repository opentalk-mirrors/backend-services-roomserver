# Command to determine the current Health Status

Determine the health status of the service from the Command-Line interface using a subcommand.

## `opentalk-roomserver health` subcommand

Help output looks like this:

<!-- begin:fromfile:cli-usage/health-help.md -->

```text
Return the readiness state

Usage: opentalk-roomserver health [ENDPOINT]

Arguments:
  [ENDPOINT]  The monitoring endpoint can be provided optionally

Options:
  -h, --help  Print help
```
