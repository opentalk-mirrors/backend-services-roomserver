# Command to print signaling module information

This command prints either a list or detailed description about all available modules.

## `opentalk-roomserver modules` subcommand

Help output looks like this:

<!-- begin:fromfile:cli-usage/modules-help.md -->

```text
Command show available modules

Usage: opentalk-roomserver modules <COMMAND>

Commands:
  list                 List available modules and their features
  print-documentation  Print a documentation of the modules (in Markdown)
  help                 Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
