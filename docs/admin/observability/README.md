# Observability

Observability features help administrators understand the state of the {{ product_name }} RoomServer during operation. The following guides describe how to configure each capability:

- [Logging](logging.md) explains how to adjust log verbosity via environment variables.
- [Metrics](metrics.md) covers exposing Prometheus-compatible metrics and configuring access control.
- [Monitoring](monitoring.md) describes the built-in health service and its `UP`/`READY` status responses.
- [Tracing](tracing.md) details exporting OpenTelemetry traces to an OTLP endpoint.
